# next-touch-events
> Next touch events for different devices

## install
```bash
npm install -S afeiship/next-touch-events --registry=https://registry.npm.taobao.org
```

## apis
| api   | description   |
| ----- | ------------- |
| START | pointer-start |
| MOVE  | pointer-move  |
| END   | pointer-end   |

## usage
```js
import NxTouchEvents from 'next-touch-events';

// code goes here:
NxTouchEvents.START
NxTouchEvents.MOVE
NxTouchEvents.END
```
